# JD 商品提取 — 用户使用指南

本扩展用于在京东网站上**一键导出商品信息**，保存为 `.jsonl` 文件（可用 Excel、Python 等工具打开处理）。

---

## 一、安装扩展

1. 获取本项目的文件夹（需包含 `manifest.json`）。
2. 打开浏览器扩展页面：
   - **Chrome**：地址栏输入 `chrome://extensions`
   - **Edge**：`edge://extensions`
3. 打开右上角 **「开发者模式」**。
4. 点击 **「加载已解压的扩展程序」**，选中项目文件夹。
5. 把扩展图标 **固定到工具栏**，方便以后点击。

> 安装后**不需要**运行任何命令或安装 Node.js，直接在京东网页上使用即可。

---

## 二、能用在哪些页面？

| 页面类型 | 网址示例 | 能做什么 |
|----------|----------|----------|
| 商品详情 | `https://item.jd.com/1234567890.html` | 提取**完整**商品信息（标题、价格、图片、规格、详情等） |
| 搜索结果 | `https://search.jd.com/Search?keyword=手机` | 提取**列表摘要**或批量进入每个商品抓详情 |
| 列表页 | `https://list.jd.com/...` | 同搜索页 |

**不能用的页面**：购物车、订单、个人中心、非京东商品页等。

---

## 三、提取单个商品（最常用）

1. 用浏览器打开要导出的**京东商品页**，等页面完全加载。
2. 任选一种方式：
   - 点击工具栏上的扩展图标，再点 **「提取并下载 JSONL」**；
   - 或点击页面**右下角红色按钮**「提取并下载 JSONL」。
3. 扩展会**自动向下滚动**页面（请保持该标签页在前台，不要切走），滚动结束后开始提取。
4. 浏览器会下载一个 `.jsonl` 文件，文件名类似：`jd-products-2026-05-26-1234567890.jsonl`。

### 其他按钮（商品页）

| 按钮 | 作用 |
|------|------|
| **只提取商品信息** | 只保存到扩展内部缓存，**不下载文件** |
| **复制 JSON** | 把刚提取的内容复制到剪贴板 |
| **下载已缓存 JSONL** | 把之前提取过的**全部**商品一次性导出 |
| **清空 JSONL 缓存** | 删除扩展里保存的所有商品记录 |

## 四、提取搜索结果

在**搜索页**打开扩展后，界面会多出「翻页页数」「详情间隔」等设置，页面右下角会出现 **①～⑦** 一排彩色按钮。

### 先搞清两种需求

| 你的目标 | 推荐操作 |
|----------|----------|
| 只要列表里的标题、价格、主图等**简要信息** | **② 翻页提取 JSONL** |
| 要每个商品的**完整详情**（规格、长描述、多规格价格等） | **⑦ 翻页逐一点开详情提取** |

列表提取**快**；完整详情**慢**（每个商品都要打开详情页），请耐心等待。

### 设置说明

在扩展弹窗里可以调整：

- **自动翻页页数**：填 `1` 只抓当前页；填 `5` 会自动点「下一页」最多抓 5 页。
- **详情页间隔**：批量抓详情时，每个商品之间等待的毫秒数。数字越大越慢，但更不容易被网站限制。**建议不少于 2500**。

### 右下角按钮速查

| 按钮 | 适合场景 |
|------|----------|
| **① 提取本页 JSONL** | 只要当前这一页的商品摘要 |
| **② 翻页提取 JSONL** | 多页商品摘要，自动翻页 |
| **③ 本页缓存商品链接** | 先把链接存起来，稍后再抓详情 |
| **④ 翻页缓存商品链接** | 多页链接一次性存进队列 |
| **⑤ 批量打开详情并提取** | 对③④存好的链接，逐个打开详情并提取 |
| **⑥ 本页逐一点开详情提取** | 当前页：先列表，再逐个进详情抓完整信息 |
| **⑦ 翻页逐一点开详情提取** | 多页 + 逐个详情，**最完整也最耗时** |

### 使用 ⑦ 时要注意

1. 打开搜索页，设好翻页数和间隔。
2. 点 **⑦**，**不要关闭搜索页标签**，尽量保持它在当前窗口。
3. 右下角会弹出进度提示；全部完成后会自动下载汇总文件。
4. 商品很多时可能跑**几十分钟到几小时**，请勿中途关浏览器。

### 分两步：先存链接、晚上再抓

1. 白天：点 **④ 翻页缓存商品链接**。
2. 晚上：点 **⑤ 批量打开详情并提取**。

「清空链接队列」只删待抓链接，**不会**删掉已经提取好的商品数据。

## 五、下载的文件是什么？

- 格式：**JSONL**（每行一条商品记录）。
- 可用记事本打开；用 Python 的 `pandas.read_json(..., lines=True)` 或在线 JSONL 查看器更方便。
- 多条商品 = 文件里有多行，每行是一个 `{ ... }` 商品对象。

常见文件名：

| 场景 | 文件名示例 |
|------|------------|
| 单个商品 | `jd-products-2026-05-26-1234567890.jsonl` |
| 搜索列表 | `jd-search-2026-05-26-手机.jsonl` |
| 深度批量 | `jd-deep-crawl-2026-05-26.jsonl` |

## 六、常见问题

**提示「请先打开京东商品详情页 / 搜索页」**  
→ 先点到正确的京东页面，再点扩展图标（扩展看的是**当前正在看的标签页**）。

**一直转圈然后超时**  
→ 商品页：确保页面在前台、已加载完，刷新后重试。  
→ 搜索/批量：减少翻页页数，或把「详情间隔」调大。

**搜不到商品 / 列表是空的**  
→ 等搜索结果出来后再点提取；或先手动滚一下页面让商品加载出来。

**价格显示不出来**  
→ 部分商品需要**登录京东**后才能看到价格。

**重装扩展后以前的数据没了**  
→ 数据存在浏览器里，卸载会清空。重要结果请用 **「下载已缓存 JSONL」** 及时备份。

**抓太多会不会被封？**  
→ 请合理设置间隔、不要短时间大量抓取；本工具仅供个人学习研究，请遵守京东网站使用规则。

## 七、推荐使用方式小结

```
只导 1 个商品：
  打开商品页 → 「提取并下载 JSONL」

要很多商品的简介（标题、价、图）：
  打开搜索页 → ② 翻页提取 JSONL

要很多商品的完整信息：
  打开搜索页 → 设翻页和间隔 → ⑦ 翻页逐一点开详情提取 → 等完成
```
